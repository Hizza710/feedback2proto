## 課題名

Think from Users, Think Together - Let's Feedforward to Our Prototype - 

1年の終わりに、どのような1年だったかを振り返るとともに、次の1年への希望を集めよう。

## 課題内容（どんな作品か）

(背景)
- チームメンバーそれぞれが、プロトタイプをつくった後、多面的にコメントを得られていないまま、次のプロトタイピング開発に進んでしまうことがもったいないと感じていた
- プログラマー視点でのコメントだけでなく、ユーザー視点からのフィードバックも実装の力になるため、それを集めやすい環境をつくりたいと考えた
- チームメンバーの作品の確認が同一リンクでできるともっとみやすくなるのではと考えた
- また受け取ったコメントも一覧でみられると、毎回使いやすいと考えた

(内容)
- プロトタイプをつくったメンバーが自分でフィードバックをもらいたい観点と共に投稿
- 投稿をすると自分のプロトへコメントを集めるカードが立ち上がり、届いたコメントは一覧に
- スタンプ機能もあり、反応を集められるようにもなる、また投稿を消すことも自由にできる

## 工夫した点・こだわった点

- 機能面
    - 誰かが投稿をした内容をみて、他の誰かさらに投稿できる流れをつくった
    - 投稿は一括CSVダウンロードができるようにもした
    - 投稿後のカードを自由に消せるようにした
    - 開発環境とサーバー環境とで同じファイルで作業できるようにした

- コード面
    - 前回はphpファイルをたくさん作ると混乱するために、複数の機能も1枚の中でつくったが、前回メンターの方から現場では機能で分けるとよいよと伺ったため、今回機能毎に分けてみた。
        - **index.php:** 最初に開くページ。投稿フォームがあり、作品や意見を入力して送信できる
        - **select.php:** みんなの投稿一覧を見るページ。投稿内容やコメントが表示される。
        - **insert.php:** 投稿フォームから送信された内容をデータベースに保存するための処理ファイル。
        - **insert_comment.php:** 投稿に対するコメントをデータベースに保存するための処理ファイル。
        - **add_stamp.php:** 投稿にスタンプ（リアクション）を追加するための処理ファイル。
        - **reset_db.php:** データベースを初期化（リセット）するためのファイル。
        - **export_csv.php:** 投稿データをCSVファイルとしてダウンロードできるファイル。
        - **db_conn.php:** データベースと接続するための設定ファイル。
        - **config.sample.php:** データベース接続情報などの設定ファイル。本番用はgitignoreとして、config.sample.phpをサンプルとしていれた。
        - **lib.php: 投稿へのスタンプと、荒らし帽子機能をいれたファイル。**
